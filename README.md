# Sales Analytics Dashboard

![Sales Analytics Dashboard](preview.png)

An interactive **sales analytics dashboard** built with Streamlit. Explore sales performance by category, region, and time with beautiful visualizations.

## ✨ Features

- Interactive filters (Category, Region, Date Range)
- Key business metrics (Total Sales, Orders, Average Order Value)
- Sales distribution by Category and Region (Pie charts)
- Monthly sales trend analysis (Line chart)
- Clean and responsive design

## 🚀 Live Demo

👉 [View Live Dashboard](https://somtocodes-web-analysis.streamlit.app/)

## 🛠 Tech Stack

- **Streamlit** – Web framework
- **Pandas** – Data processing & cleaning
- **Plotly** – Interactive charts

## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/somtocodes/Streamlit-analysis.git
cd Streamlit-analysis

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run analysis.py
